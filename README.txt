This is a free extensible plugin to add online payment to your site!

== Description ==

This plugin provides a payment form using shortcodes. You can put the shortcode in your posts or pages and enable payment in your website.
**MANA Gateway** is an independent plugin and functions on its own.
This is the first version and it contains 1 payment gateway but **MANA Gateway** is extensible and you can write any gateway for it.

== Installation ==

1. Upload `mana-gateway` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
